| Command | Description |
| ------- | ----------- |
| ```vagrant up``` | Build the environment as specified by ./Vagrantfile |
| ```vagrant ssh [<hostname>]``` | SSH into a host. If there is only one host specified in .\Vagrantfile, the hostname is optional  |
